Making Video Games

I have always loved video games since I was little.
Today I create it for fun.
I am going to tell you about a tool that I use for the creation of video games, it is Unreal engine!

What is Unreal Engine ?

Unreal engine is a game engine. A game engine is: a system manager (input / output, user interface, memory, etc.), a graphics engine (2D or +), a sound engine, possibly a network manager (for multiplayer games) , a physics engine and possibly tools for artificial intelligence. In short, a game engine is a collection of specialized engines required to make a game.
